Dice Roller C#, Github, Visual Studio, OOP

 

  This is a C# console app that simulates dice rolling with simple graphics. Users start by picking a dice color and specifying how many sides they want the dice to have. After that users can then roll the selected dice on a table and select which dice they want to keep and which ones they want to roll again.

Virtual Pet C#, Github, Visual Studio, OOP

 

  This is another C# console app where users interact with a shelter of pets. Users can add their own pets to the shelter and choose from the categories of organic or robotic pets. All pets in the shelter will need to be taken care of via interactions (play, feed, take to doctor). Each pet has levels of health, boredom, and hunger that will deteriorate as time passes by if not taken care of. They will start making alert noises if not taken care of properly. Interactions come with sound effects and text art. Finally, pets can be adopted out of the shelter.
